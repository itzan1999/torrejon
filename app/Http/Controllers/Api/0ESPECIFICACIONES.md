# ESPECIFICACIONES ACERCA DEL NOMBRAMIENTO DE LOS MÉTODOS DE LOS CONTROLADORES

## ESPECIFICACIONES

## OBTENER DATOS

1. Para listar todo se debe nombrar de la siguiente manera: **index()**

2. Para listar un elemento filtrado por id se debe nombrar de la siguiente manera: **show()**

## CREAR DATOS

1. Para crear un nuevo elemento se debe nombrar de la siguiente manera: **store()**

## BORRAR DATOS

1. Para borrar un elemento por id se debe nombrar de la siguiente manera: **destroy()**

## ACTUALIZAR DATOS

1. Para actualizar un elemento por id se debe nombrar de la siguiente manera: **update()**
